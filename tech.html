Website Examples

Completed over 500 hours of training in Website Development at UC Irvine in which I learned the full stack, with the skills needed to be a proficient front-end and back-end developer. I enjoy building the front-end, and with my illustration skills have an eye for detail. In addition, knowing the back-end helps in seeing how it all ties and relates together. What good is a beautifully designed website if it doesn't work properly, or a perfectly working website that has a terrible design? Both are equally important! Below are some of the projects I and/or a team created. I worked on either the front-end or the back-end. This page will continually be updated to reflect new work.

Squpon Coupon Finder

An API powered application that searches for available online or in-store coupons, deals, and discounts from the user's search parameters by city location or type.
Built with HTML, CSS, Materialize, Ajax, Google Map & Sqoop API's.

The BarKeeper

An online cocktail recipe database in which a user can find drink recipes from a search using liquor type, drink name or ingredient. In addition, a user can add a recipe into the database.
Built with HTML, CSS, Boostrap, Ajax, theCocktailDB API, Node.js, Sequelize, Express, MySQL

Eat da Burger

Eat-Da-Burger! is a restaurant app that lets users input the names of burgers they'd like to eat. The app displays the orders in the UI; allow users to "devour" them when ready and store every burger in a database, whether devoured or not.
Built with HTML, CSS, Boostrap, Node.js, Express, Method-Override, Body-Parser, Handlebars & MySQL.

Friend Finder

A compatibility-based "FriendFinder" application. This full-stack site takes in user survey results, then compares the answers with other users. It then displays the name and picture of the user with the best overall match.
Built with HTML, CSS, Boostrap, Node.js, Express, NPM Body-Parser & Path.

Giphy Search

An API was used to make this dynamic web page that populates with gifs of your choice. JavaScript and jQuery was used to change the HTML dynamically.
Built with HTML, CSS, Boostrap, Javascript, Jquery, Ajax, and Giphy API.
